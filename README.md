# IrishTrainTimes_Privacy-Policy
Privacy policy for IrishTrainTimes

## Privacy Policy

This app collects and stores no data from the user.
IrishTrainTimes was built as a free app. This SERVICE is provided at no cost and is intended for use as is.

## Permission
IrishTrainTimes requires access to the devices location in order to improve the apps functionability but this data is not collected nor stored by the app.

Thus, you are advised to review this page periodically for any changes

These changes are effective immediately after they are posted on this page
